# Click Counter App

A simple web app that counts your button clicks and saves the count using **Local Storage**.

## Features
- Increase and reset button clicks  
- Saves count even after refresh using local storage  
- Simple and responsive UI  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Use
1. Open the `index.html` file in your browser.  
2. Click the **Increase** button to add to the count.  
3. Click **Reset** to set it back to zero.
